# CodeCoach AI 🤖

An AI-powered programming feedback tool that analyzes student code using Flask, SQLite, and a local LLM.

## Features

- Automated code quality analysis
- AI-generated explanations using Llama 3.2
- Complexity analysis
- Suggested improvements
- Submission history tracking

## Tech Stack

- Python
- Flask
- SQLite
- Ollama / Llama 3.2
- HTML/CSS/Jinja2
